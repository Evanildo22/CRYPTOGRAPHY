Secure File Share

Hybrid encryption with two key-exchange modes, digital signatures, perfect forward secrecy, and a tamper-evident audit log.

Mode A — RSA Key Transport

A random AES-256 session key is wrapped with the recipient's RSA-2048 public key (OAEP). Only the matching private key can decrypt the file.

Mode B — ECDH + PFS

An ephemeral P-256 keypair is generated, ECDH + HKDF derive the AES key, and the ephemeral private key is immediately destroyed. Compromise of the long-term key cannot expose past sessions.

Signatures

Every ciphertext is signed with RSA-PSS-SHA-256. Verification is the first operation on download — any tampering aborts the request.

Audit Log

Every security event is recorded with an HMAC-SHA256 tag. Edits to past entries are detectable without the server secret.
